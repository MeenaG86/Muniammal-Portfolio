⚙️ How It Works
The portfolio is built using semantic HTML and styled with CSS for a modern look.
JavaScript is used to:
Toggle menu in mobile view
Handle contact form validation
Enable clickable actions (email, phone, location)
Add interactive UI behavior

📬 Contact Form

The contact form includes:

Input validation using JavaScript
Error and success messages
Form reset after submission

Note: This form currently works on frontend only (no backend integration).
